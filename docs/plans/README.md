# Plans

This directory keeps planning snapshots that explain why the repo moved in its
current direction.

- [verifier-first-original-plan.md](verifier-first-original-plan.md) is the
  original verifier-first plan that was previously kept as an untracked
  `pro-plan.md` local note.
- [verifier-first-executed-plan.md](verifier-first-executed-plan.md) is the
  adjusted plan actually executed in this repo after the first PR sequence and
  the current documentation reconciliation.

These files are historical planning records, not the active operator runbook.
Use [../architecture/deployable-unit.md](../architecture/deployable-unit.md)
and [../architecture/reconstruction-plan.md](../architecture/reconstruction-plan.md)
for the current assembly map and next implementation path.
