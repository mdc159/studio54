MEMORANDUM

SUBJECT: Artificial Intelligence Tool Usage Policy

1. SCOPE
   1.1 Applies to all [Company Name] employees using AI tools
   1.2 Covers AI-assisted code generation, text editing, and productivity tools
   1.3 Effective immediately upon distribution

2. PERMITTED USES
   2.1 Authorized AI Tools:
      a) GitHub Copilot Business for code generation
      b) Only tools pre-approved by IT and Legal
   2.2 GitHub Copilot Usage Guidelines:
      a) Use only for code suggestions, not entire function/class generation
      b) Developers must modify >50% of AI-suggested code
      c) Immediately remove any suggested code with third-party license headers
   2.3 Permissible AI Text Generation:
      a) Sales team may use for email drafting
      b) Must manually edit and verify originality
      c) Prohibited from using for customer-facing legal or contractual documents

3. PROHIBITED USES
   3.1 Confidential Information Restrictions (Ref: Database Schema Incident):
      a) Do not input any customer data, PII, or internal schemas into external AI services
      b) Prohibit copying source code, architectural diagrams, or financial projections
   3.2 Intellectual Property Protections (Ref: Marketing Copy Incident):
      a) Do not reproduce verbatim AI-generated text
      b) Verify originality of all AI-generated content
      c) Immediately report potential copyright infringements
   3.3 Licensing Compliance (Ref: GPL License Header Incident):
      a) Do not commit AI-generated code without thorough license review
      b) Remove any automatically inserted third-party license headers
      c) Consult Legal for any uncertain licensing scenarios

4. ENFORCEMENT
   4.1 Violation Consequences:
      a) First Violation: Mandatory compliance training
      b) Repeated Violations: Performance review discussion
   4.2 Compliance Monitoring:
      a) Quarterly random code and document reviews
      b) Incident reporting through existing company communication channels
   4.3 Legal Acknowledgment:
      a) AI-generated content may not be independently copyrightable
      b) Employee remains responsible for final work product
