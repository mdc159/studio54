1. SCOPE
   1.1 This policy applies to all employees of [Company Name]
   1.2 Defined AI Tools: Software systems using machine learning to generate, complete, or suggest content, including:
      a) Code generation assistants
      b) Text prediction and generation tools

   1.4 Supersedes all previous informal practices regarding AI tool usage

   2.3 AI-Generated Content Requirements:
      a) Mandatory line-by-line review documenting substantive human modifications
      b) Verification of copyright and original work status

   2.4 Confidential Information Exclusions:

3. PROHIBITED USES

      c) Using AI-generated text without explicit copyright verification
      d) Incorporating third-party license headers without Legal review
      e) Reproducing verbatim AI-generated content without modification
   3.2 Compliance Restrictions:
      a) Using unapproved AI tools
      b) Actions compromising SOC2, GDPR, or FedRAMP compliance

      c) Annual review of approved AI tools by IT and Legal
   4.3 Reporting Mechanism:
      a) Mandatory reporting of potential violations through existing incident management system
      b) Each department will designate an AI compliance liaison
   4.4 Intellectual Property Clarification:
      a) AI-generated content is not automatically copyrightable
      b) Employees must document AI tool contribution to any created work
      c) Company retains right to determine IP ownership of AI-assisted work
