MEMORANDUM

SUBJECT: Artificial Intelligence Tool Usage Policy

1. SCOPE
   1.1 This policy applies to all employees of [Company Name]
   1.2 Covers all AI tools, including:
      a) Code generation and assistance
      b) Text generation and editing
      c) Conversational and productivity tools
   1.3 Effective immediately upon distribution
   1.4 Supersedes all previous informal practices regarding AI tool usage

2. PERMITTED USES
   2.1 Authorized AI Tools:
      a) GitHub Copilot Business (code generation)
      b) Platforms added to Approved AI Tools List via formal IT/Legal review process
   2.2 Engineers may use GitHub Copilot for code completion and suggestion purposes
   2.3 Sales team may use approved AI tools for email drafting
   2.4 AI-Generated Content Requirements:
      a) Mandatory line-by-line review documenting substantive human modifications
      b) Tracking sheet capturing AI tool source, generated content, and human edits
      c) Verification of original work status before final submission
   2.5 Confidential Information Definition:
      a) Customer PII as defined in current Data Protection Addendum
      b) Proprietary source code
      c) Architectural diagrams
      d) Financial projections
      e) Unreleased product specifications

3. PROHIBITED USES
   3.1 Specific Prohibited Actions (Ref: Base Fact Incidents):
      a) Inputting defined confidential information (Section 2.5) into any external AI service
      b) Generating complete work products without documented manual review
      c) Using AI-generated text without copyright verification
      d) Incorporating third-party license headers without Legal review
      e) Reproducing verbatim AI-generated content without modification
   3.2 Compliance Violations:
      a) Using unapproved AI tools
      b) Actions compromising SOC2, GDPR, or FedRAMP compliance

4. ENFORCEMENT
   4.1 Violation Classification:
      a) Minor Violation: Unreported AI tool use, incomplete review documentation
         - Consequence: Written warning, mandatory retraining
      b) Moderate Violation: Inputting restricted information, copyright infringement
         - Consequence: Performance improvement plan, potential financial liability
      c) Major Violation: Repeated non-compliance, intentional policy circumvention
         - Consequence: Potential termination, legal action
   4.2 Compliance Oversight:
      a) Managers track team AI tool usage via existing project management tools
      b) Quarterly compliance review using current access and collaboration logs
   4.3 Reporting Mechanism:
      a) Mandatory reporting of potential violations through existing incident management system
      b) Department AI compliance liaisons identified in quarterly team meetings
   4.4 Intellectual Property Clarification:
      a) All AI-generated content remains subject to company IP assignment agreements
      b) Employees must document AI tool contribution to any created work
