MEMORANDUM

SUBJECT: Artificial Intelligence Tool Usage Policy

1. SCOPE
   1.1 This policy applies to all employees of [Company Name]
   1.2 Covers all AI tools, including but not limited to generative text, code generation, and conversational AI platforms
   1.3 Effective immediately upon distribution
   1.4 Supersedes all previous informal practices regarding AI tool usage

2. PERMITTED USES
   2.1 GitHub Copilot Business is the sole authorized AI coding assistant
   2.2 Engineers may use GitHub Copilot for code completion and suggestion purposes
   2.3 Sales team may use AI tools for email drafting using company-approved platforms
   2.4 All AI-generated content must be reviewed and substantively edited by the employee before use
   2.5 Permitted tools must not require input of company confidential or customer-identifying information

3. PROHIBITED USES
   3.1 Inputting customer personally identifiable information (PII) into any external AI service (Ref: Base Fact #2, Incident #1)
   3.2 Using AI tools to generate complete code blocks without manual review and modification
   3.3 Copying AI-generated text or code without verifying original copyright status (Ref: Base Fact #2, Incident #2)
   3.4 Using AI tools that are not explicitly approved by IT and Legal departments
   3.5 Inputting proprietary company source code or architectural diagrams into external AI services
   3.6 Accepting AI-generated code with third-party license headers without explicit legal review (Ref: Base Fact #2, Incident #3)
   3.7 Using AI tools in ways that could compromise SOC2, GDPR, or pending FedRAMP compliance

4. ENFORCEMENT
   4.1 Violations will result in immediate disciplinary action up to and including termination
   4.2 Managers are responsible for monitoring team AI tool usage
   4.3 IT will conduct quarterly audits of AI tool usage logs
   4.4 Employees must report potential policy violations to their direct manager or HR immediately
   4.5 Each department will designate an AI compliance liaison
