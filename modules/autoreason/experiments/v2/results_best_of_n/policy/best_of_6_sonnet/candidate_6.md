**MEMORANDUM**

TO: All Employees
FROM: CEO
DATE: [Date]
RE: Artificial Intelligence Tool Usage Policy

This policy governs the use of AI tools by all employees effective immediately.

## SCOPE

1. This policy applies to all 200 employees across engineering, sales, and other departments.
2. AI tools include any software that uses artificial intelligence for code generation, content creation, data analysis, or automated decision-making.
3. This policy covers AI tool usage for any work-related activity, whether on company devices or personal devices for company business.

## PERMITTED USES

1. GitHub Copilot Business is approved for all 80 licensed users for code completion and generation within approved repositories.
2. AI tools may be used for internal documentation, meeting summaries, and draft communications that do not contain customer data.
3. Sales teams may use AI for email templates and proposal drafts, provided all output is reviewed before sending to customers.
4. AI-generated code must be reviewed by a senior engineer before merging to production repositories.
5. All AI-generated content must be clearly identified in code comments or document headers.

## PROHIBITED USES

1. Inputting customer PII, financial data, database schemas, or any customer-specific information into AI tools (motivated by SOC2, GDPR obligations, and Q1 database schema incident).
2. Using AI tools other than GitHub Copilot Business without prior IT approval (motivated by budget constraints and security controls).
3. Directly copying AI-generated text that may contain copyrighted material without verification (motivated by Q2 competitor copyright incident).
4. Committing AI-generated code without removing training data artifacts, license headers, or attribution text (motivated by Q1 GPL license incident).
5. Using AI tools for processing EU customer data or FedRAMP-related work (motivated by GDPR and pending FedRAMP authorization).
6. Enabling AI features in unauthorized platforms including Slack AI capabilities (motivated by existing security configuration).

## ENFORCEMENT

1. Engineering managers will conduct monthly code reviews to verify AI-generated code compliance and proper attribution.
2. IT will monitor software installations and block unauthorized AI tools through existing endpoint management.
3. Sales managers will review customer communications containing AI-generated content before delivery.
4. First violation results in mandatory security training; second violation results in written warning; third violation results in termination.
5. Data privacy violations involving customer information will result in immediate suspension pending investigation.
6. All employees must complete AI policy training within 30 days of this memo's effective date.
