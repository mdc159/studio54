**MEMORANDUM**

TO: All Employees
FROM: Executive Team
DATE: [Date]
RE: AI Tool Usage Policy

## SCOPE

1. This policy applies to all 200 employees using artificial intelligence tools for work purposes.
2. Covers all AI-powered software, services, and applications used to generate, modify, or analyze content including code, text, images, and data.
3. Applies to both company-provided and personal AI tools used for business activities.

## PERMITTED USES

1. GitHub Copilot Business for code generation and assistance (80 licensed seats available).
2. AI tools for generating internal documentation, meeting notes, and non-customer-facing content.
3. AI assistance for brainstorming, research, and ideation using publicly available information only.
4. Code refactoring and optimization suggestions through approved tools.
5. Draft creation for internal communications and proposals.

## PROHIBITED USES

1. Inputting customer PII, financial data, database schemas, or any customer information into AI tools (motivated by SOC2/GDPR obligations and DPA violations from database schema incident).
2. Using unapproved AI tools or services outside GitHub Copilot Business (motivated by security controls for FedRAMP authorization).
3. Copying AI-generated content without verification for originality and licensing compliance (motivated by sales rep copyright incident and GPL license code commit).
4. Submitting proprietary code, algorithms, or confidential business information to external AI services (motivated by IP protection and compliance requirements).
5. Using AI-generated content in customer-facing materials without legal review (motivated by copyright infringement risk).
6. Bypassing existing code review processes for AI-generated code commits (motivated by GPL license incident).

## ENFORCEMENT

1. Engineering managers will verify GitHub Copilot usage during existing code reviews and ensure no prohibited content was input.
2. IT will monitor network access to unapproved AI services through existing firewall logs and block unauthorized tools.
3. Sales leadership will review customer communications during monthly pipeline reviews to identify potential AI-generated content violations.
4. First violation results in mandatory training and documentation in employee file.
5. Second violation results in written warning and temporary AI tool access suspension.
6. Third violation results in termination.
7. Immediate termination for violations involving customer data or confidential information.
8. All employees must acknowledge this policy within 30 days of publication.
