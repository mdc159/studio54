# MEMORANDUM

**TO:** All Employees  
**FROM:** Executive Team  
**DATE:** [Current Date]  
**RE:** AI Tool Usage Policy

## SCOPE

1. This policy applies to all employees using artificial intelligence tools for work-related activities.
2. Covered tools include code generation assistants, text generation services, and AI-enabled features in business applications.
3. This policy governs use of both company-provided and personal AI tools for business purposes.

## PERMITTED USES

1. GitHub Copilot Business for code generation and completion within approved development environments.
2. AI tools for internal documentation, meeting summaries, and non-customer-facing content creation.
3. AI assistance for code refactoring and optimization of internally-developed, non-proprietary algorithms.
4. AI-generated content for marketing materials, provided all output undergoes legal review before publication to prevent copyright infringement issues identified in prior incidents.

## PROHIBITED USES

1. Inputting customer PII, financial data, or any information covered by Data Processing Agreements into non-approved AI services (motivated by SOC2, GDPR obligations, and Q1 database schema incident).
2. Using AI tools other than GitHub Copilot Business for code generation without prior IT approval (motivated by GPL license contamination incident and pending FedRAMP authorization requirements).
3. Submitting proprietary database schemas, API specifications, or system architecture details to external AI services (motivated by Q1 customer data exposure incident).
4. Publishing AI-generated content externally without legal review to verify originality (motivated by competitor copyright infringement incident).
5. Using personal AI tool accounts or free-tier services for any work-related activities (motivated by SOC2 compliance and data control requirements).
6. Incorporating AI-generated code with incompatible licensing terms into company products (motivated by GPL contamination incident and legal counsel guidance on copyrightability).

## ENFORCEMENT

1. All GitHub Copilot usage will be monitored through existing license management and code review processes.
2. Marketing content undergoes mandatory legal review before publication through current approval workflows.
3. Code commits containing AI-generated content require explicit disclosure in pull request descriptions for review.
4. Violations result in immediate revocation of AI tool access and disciplinary action up to termination.
5. IT will conduct quarterly access reviews using existing user provisioning systems to ensure compliance.
6. Employees must report suspected policy violations to their manager or legal team within 24 hours of discovery.
