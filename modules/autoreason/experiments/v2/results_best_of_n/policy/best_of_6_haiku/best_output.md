MEMORANDUM

Subject: Artificial Intelligence Tool Usage Policy
Effective Date: [Current Date]

1. SCOPE
   1.1 This policy applies to all employees of [Company Name]
   1.2 Covers all AI tools and generative AI services, including but not limited to large language models, coding assistants, and content generation platforms
   1.3 Applies to company-owned and personal devices used for work purposes

2. PERMITTED USES
   2.1 GitHub Copilot Business is authorized for code generation and development assistance
   2.2 AI tools may be used for:
      a) Code completion and suggestion
      b) Writing documentation
      c) Drafting internal communication
      d) Generating preliminary marketing and sales content
   2.3 Users must review and validate all AI-generated output before use
   2.4 Users must remove or anonymize any personally identifiable information (PII) before AI tool input

3. PROHIBITED USES
   3.1 Prohibited Input Categories (Ref: Base Fact #2, Incident #1)
      a) Customer database schemas
      b) Full customer records
      c) Unredacted PII
      d) Confidential financial information
      e) Proprietary source code

   3.2 Prohibited Output Uses (Ref: Base Fact #2, Incident #2)
      a) Verbatim reproduction of third-party copyrighted material
      b) Direct submission of AI-generated content without substantive human modification
      c) Using AI tools to circumvent intellectual property protections

   3.3 Licensing and Compliance Restrictions (Ref: Base Fact #2, Incident #3)
      a) Do not use AI-generated code with potentially incompatible open-source licenses
      b) Do not commit code containing embedded third-party license headers
      c) Verify license compatibility for all AI-assisted code

   3.4 Unauthorized Tools
      a) No use of personal or unsanctioned AI services
      b) No input of company data into non-approved AI platforms

4. ENFORCEMENT
   4.1 Violations will result in progressive disciplinary action
   4.2 First violation: Written warning
   4.3 Second violation: Performance improvement plan
   4.4 Third violation: Potential termination
   4.5 Mandatory reporting of AI tool usage incidents to legal and compliance teams
   4.6 Quarterly review of AI tool usage by IT security and legal departments

Approved: [CEO Signature]
Prepared by: Legal and IT Security Teams
