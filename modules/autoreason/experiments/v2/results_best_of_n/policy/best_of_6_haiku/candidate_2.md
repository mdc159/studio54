MEMORANDUM

Subject: Artificial Intelligence Tool Usage Policy
Effective Date: [Current Date]

1. SCOPE
   1.1 This policy applies to all employees of [Company Name]
   1.2 Covers all AI tools, including but not limited to generative AI, coding assistants, and language models
   1.3 Applies to company-owned and personal devices used for work purposes
   1.4 Supersedes all prior informal practices regarding AI tool usage

2. PERMITTED USES
   2.1 GitHub Copilot Business is the sole approved AI coding assistant
   2.2 Engineers may use GitHub Copilot for code completion, refactoring, and documentation
   2.3 Sales team may use AI tools for email drafting and customer communication templates
   2.4 AI tools must be used only through company-licensed accounts
   2.5 Users must review and validate all AI-generated content before use
   2.6 All AI-generated work requires explicit human oversight and modification

3. PROHIBITED USES
   3.1 No AI tool usage is permitted with customer personally identifiable information (PII) [Ref: Base Fact - Incident 1]
   3.2 Prohibited to input any customer data, database schemas, or confidential information into external AI services [Ref: Base Fact - Incident 1]
   3.3 Do not use AI-generated text without verifying original authorship and copyright clearance [Ref: Base Fact - Incident 2]
   3.4 Strict prohibition on committing AI-generated code with third-party license headers [Ref: Base Fact - Incident 3]
   3.5 No use of personal or unauthorized AI accounts for work purposes
   3.6 AI tools cannot be used for tasks requiring regulatory compliance certification
   3.7 Prohibited from using AI tools that are not explicitly approved by IT and Legal departments

4. ENFORCEMENT
   4.1 Violations will result in immediate disciplinary action, up to and including termination
   4.2 Mandatory quarterly training on AI tool usage policy
   4.3 Code reviews will include verification of AI tool usage compliance
   4.4 Random audits will be conducted of AI-generated work
   4.5 Employees must report potential policy violations immediately
   4.6 Compliance will be tracked through existing performance management systems
