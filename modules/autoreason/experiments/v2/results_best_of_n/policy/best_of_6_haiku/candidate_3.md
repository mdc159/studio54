AI TOOL USAGE POLICY
Effective Date: [Current Date]

SCOPE
1. This policy applies to all employees of [Company Name], regardless of department or role.
2. The policy covers all generative AI tools, including but not limited to large language models, code generation assistants, and AI-powered writing tools.
3. The policy applies to company-owned and personal devices used for work purposes.
4. This policy supplements existing information security and intellectual property guidelines.

PERMITTED USES
1. GitHub Copilot Business is authorized for code generation, subject to the following restrictions:
   a. Use only on company-licensed seats
   b. Review and validate all AI-generated code before implementation
2. AI tools may be used for:
   a. Code completion and suggestion
   b. Draft writing and editing of non-confidential communications
   c. Brainstorming and ideation
3. Employees must use only tools approved by the IT Security team.
4. AI tool usage must comply with existing confidentiality and data protection agreements.

PROHIBITED USES
1. Do not input any customer personally identifiable information (PII) into AI tools (Ref: Incident 1, GDPR obligations).
2. Do not paste source code, database schemas, or technical specifications from internal systems into external AI services (Ref: Incident 1).
3. Do not use AI-generated text without substantial human review and modification (Ref: Incident 2).
4. Do not commit AI-generated code without:
   a. Verifying complete license compliance
   b. Removing any pre-existing license headers
   c. Conducting a full code review (Ref: Incident 3)
5. Do not use AI tools for:
   a. Generating final customer-facing documents
   b. Creating legal or contractual language
   c. Producing unmodified technical documentation
6. Prohibited AI tools include any not explicitly approved by IT Security.

ENFORCEMENT
1. Violations will be addressed through existing performance management processes.
2. First offense: Written warning and mandatory AI tool usage training.
3. Subsequent offenses may result in:
   a. Temporary revocation of AI tool access
   b. Disciplinary action up to and including termination
4. Managers are responsible for:
   a. Monitoring team AI tool usage
   b. Reporting potential policy violations
5. IT Security will conduct quarterly audits of AI tool usage and compliance.
6. Employees must report potential policy violations immediately to their manager and IT Security.
