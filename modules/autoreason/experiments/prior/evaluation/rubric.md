# Evaluation Rubric

Rate each proposal on the following dimensions (1-5).

## Dimensions

### Solves the Stated Problem (1-5)
1. Does not address the core problem
2. Partially addresses it, missing major aspects
3. Addresses the problem but with notable gaps
4. Addresses the problem well with minor gaps
5. Fully and directly addresses every aspect of the stated problem

### Completeness (1-5)
1. Missing most required elements
2. Covers some elements, significant omissions
3. Covers most elements, a few gaps
4. Covers all elements, minor areas could be deeper
5. Comprehensive — all elements covered with appropriate depth

### Coherence (1-5)
1. Contradictory or disorganized
2. Some internal contradictions or unclear structure
3. Generally coherent with minor inconsistencies
4. Well-organized, internally consistent
5. Tightly coherent — every piece supports and connects to the whole

### Absence of Unnecessary Complexity (1-5)
1. Overwrought — complexity far exceeds what the problem requires
2. Noticeably over-engineered in several areas
3. Some unnecessary complexity but mostly appropriate
4. Appropriately scoped with minimal excess
5. Lean — nothing that doesn't earn its place

## Instructions

- You will see two proposals labeled "Proposal X" and "Proposal Y"
- Score each independently on all four dimensions
- You do NOT know which is the original and which is the reviewed version
- Score based on the proposal's merit, not on comparison between the two
- Add brief notes explaining any score of 1-2 or 5
