## Prohibited Claims & Guidance

### Legal & Compliance Risks - NEVER CLAIM:

❌ **"100% secure" or "unhackable"**
- **Risk:** Creates legal liability and unrealistic expectations
- **Instead:** "Provides enterprise-grade security controls with complete data sovereignty"

❌ **"Guarantees compliance with [specific regulation]"**
- **Risk:** Compliance is customer responsibility, creates legal exposure
- **Instead:** "Designed to support compliance requirements with features including..."

❌ **"Eliminates all security vulnerabilities"**
- **Risk:** No tool can eliminate all risks, creates false expectations
- **Instead:** "Significantly reduces security vulnerabilities through AI-powered analysis"

❌ **"Never makes mistakes" or "Zero false positives"**
- **Risk:** All AI systems have error rates
- **Instead:** "Continuously improves accuracy through machine learning and customer feedback"

### Technical Accuracy - AVOID OVERSTATING:

❌ **"Replaces human code reviewers"**
- **Risk:** Undermines professional expertise, creates unrealistic expectations
- **Instead:** "Augments and accelerates human code review processes"

❌ **"Works perfectly with all programming languages"**
- **Risk:** Performance varies by language and framework
- **Instead:** "Optimized for major enterprise programming languages with continuous expansion"

❌ **"Fastest deployment in market"**
- **Risk:** On-premise typically requires longer deployment than cloud
- **Instead:** "Comprehensive deployment support ensuring enterprise security from implementation"

### Market Positioning - BE PRECISE:

❌ **"Only secure AI code review solution"**
- **Risk:** Other solutions have security features, though cloud-based
- **Instead:** "Only AI code review solution providing complete data sovereignty"

❌ **"Best AI code review tool"**
- **Risk:** Subjective claim without supporting evidence
- **Instead:** "Leading AI code review solution for security-conscious enterprises"
